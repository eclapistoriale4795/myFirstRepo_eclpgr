git init   - Initialize Directory
git status - Check for files
	--short     - Compact Mode
	?? - Untracked files
	A - Files added to stage
	M - Modified files
	D - Deleted files
git add    - Wanna add 'em?
git rm     - Delete it?
	--all or -A - All of the files in the directory
	
Confirm these changes?
git commit -m "New Change"!

Create a branch! git branch [branch name]
Type "git branch" to see your branches!
git branch -d - Delete Branch
Select a branch! git checkout [branch name]

Want to put it up on GitHub?
git remote add origin [GitHub link to .git]
git branch -M [your branch] (Example: git branch -M master)
git push -u origin [your branch]
